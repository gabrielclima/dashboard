Versão 0.4.4:

- Controle de acesso baseado no perfil do usuário:

Por padrão têm acesso: Admin e Super-Admin.

Para configurar basta setar os direitos no campo perfil em Administração->Perfis, selecione o perfil e a 
aba Administração, campo Perfis.

Sem acesso: usuário não vai acessar o Dashboard.
Acesso para Ler ou Gravar: permite ao usuário ver o Dashboard.

- Correção de pequenos bugs.



Version 0.4.4:

- Access control based on the user's profile:

Have access by default: Admin and Super Admin.

To configure simply set the rights in the profile field in Administration-> Profiles, select the profile and the Administration tab, Profiles field.

No Access: Users will not access the Dashboard.
Read or Write access: allows the user to view the Dashboard.

- Fixed minor bugs.
